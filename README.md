# products-app-frontend

That is sample test a frontend page.

## Build

    npm install
    npm run build
    npm run start

You can build app with **Docker** using command:

    make

The site will be available at:

    http://localhost:9000

After finish working:

    make stop
    make clean

## Dependencies

* NodeJS
* npm
* make (only for Docker build)
